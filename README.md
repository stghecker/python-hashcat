# python-hashcat
A password cracker python script, based in MD5 encryption.
How to use:
1. Download python-hashcat-main.py
2. Download rockyou.txt.gz.torrent* or download from https://github.com/zacheller/rockyou (or any other wordlist)
3. Put them in the same directory
4. Run (I reccomend thronny)
*You need a torrent client (e.g. :contentReference[oaicite:1]{index=1}) to download this file, since it is distributed via torrent.
